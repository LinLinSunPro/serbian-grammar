# Vocabulary: Greetings and Time

## Time of Day (Doba Dana)

*Match the greeting to the time.*
(Source: Image 1, Ex 1)

* **Dobro jutro**: Morning (e.g., 9:00)
* **Dobar dan**: Day/Afternoon (e.g., 12:00, 15:00)
* **Dobro veče**: Evening (e.g., 18:00)
* **Laku noć**: Night (e.g., 24:00 / Ponoć)

## Key Terms

* **Podne**: Noon
* **Ponoć**: Midnight
* **Jutro**: Morning
* **Dan**: Day
* **Veče**: Evening
* **Noć**: Night

## Formal vs Informal Expressions (Formalno i Neformalno)

*Sort the expressions (Image 3, Ex 18).*

| Formalno (Formal) | Neformalno (Informal) |
| :--- | :--- |
| Doviđenja. | Ćao. |
| Kako ste? | Kako si? |
| Da li ste Vi Boris? | Da li si ti Ana? |
| Dobro veče. | Zdravo. |
| Vidimo se. | A Vi? / A ti? |
| Dobro jutro. | (A ti?) |

## Conversation Flow (Dobar dan - Image 6)

* **A**: Zdravo! Kako si?
  * **B**: Dobro smo, hvala. (Dobro sam, hvala).
* **A**: Dobar dan. Dobro sam, hvala. Kako ste Vi?
* **A**: Ćao! Nije loše. A ti?
* **A**: Ćao! Kako ste? (Informal 'Ćao' with Formal 'ste' - unusual, maybe mixed group or semi-formal).
  * **B**: Vidimo se! -> Vidimo se! Ćao!

## How are you? (Kako si / Kako ste?)

*Standard answers:*

* **Dobro sam.** (I am good.)
* **Nije loše.** (Not bad.)
* **Odlično.** (Excellent.)
* **Onako.** (So-so.)
* **Može bolje.** (Could be better.)
