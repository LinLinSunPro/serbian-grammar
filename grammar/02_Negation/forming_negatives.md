# Negative Sentences (Odrične rečenice)

*Make the sentences negative.*
(Source: Image 3, Ex 19)

1. Milica je Francuskinja. -> **Milica nije Francuskinja.**
2. Oni su Pavle i Dani. -> **Oni nisu Pavle i Dani.**
3. Ja sam stranac. -> **Ja nisam stranac.**
4. Moje ime je Elizabet. -> **Moje ime nije Elizabet.**
5. Ja se zovem Čarli. -> **Ja se ne zovem Čarli.** (Verbs other than 'biti' use 'ne' before the verb).
6. Da, jesmo. -> **Ne, nismo.**
7. Ti si moja drugarica. -> **Ti nisi moja drugarica.**
8. Loše smo. -> **Nismo loše.**
9. Da, jeste. -> **Ne, nije.**
10. Dobro sam. -> **Nisam dobro.**

## Reflexive Verbs Note

* **Zvati se**:
  * Affirmative: Ja **se** zovem.
  * Negative: Ja **se ne** zovem. (Note: 'se' position changes or stays? Standard: *Ja se ne zovem*. Questions: *Kako se zoveš?*)
