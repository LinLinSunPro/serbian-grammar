# Glagol "BITI" (Verb "To Be")

This table shows the present tense forms of the verb "to be" (Biti).

## Positive Forms (Jesam)

Includes both short (clitic) and long (emphatic) forms as indicated in your notes.

| Person | Singular | Plural |
| :--- | :--- | :--- |
| **1st (Ja / Mi)** | sam (jesam) | smo (jesmo) |
| **2nd (Ti / Vi)** | si | ste (jeste) |
| **3rd (On, Ona, Ono / Oni, One, Ona)** | je (jeste) | su |

## Negative Forms (Nisam)

| Person | Singular | Plural |
| :--- | :--- | :--- |
| **1st (Ja / Mi)** | nisam | nismo |
| **2nd (Ti / Vi)** | nisi | niste |
| **3rd (On, Ona, Ono / Oni, One, Ona)** | nije | nisu |
