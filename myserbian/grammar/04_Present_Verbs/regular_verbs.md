## Word Order (Red Reči)

*Exercise: Correct the word order.*
(Source: Image 1, Ex 5)

1. *Ja zovem se Ana.* -> **Ja se zovem Ana.** (Reflexive pronoun 'se' position)
2. *Da li govoriš ti nemački?* -> **Da li ti govoriš nemački?**
3. *Li razumete?* -> **Da li razumete?**
4. *Učim ja srpski zato što radim ovde.* -> **Ja učim srpski zato što radim ovde.** (Subject first usually)
5. *Moj je on student.* -> **On je moj student.**
6. *Da li ona je novinarka?* -> **Da li je ona novinarka?** (Enclitic placement)
7. *Sam kuvarica.* -> **(Ja) sam kuvarica.** (Subject drop is okay, but context implies declarative).
8. *Novinar nisam.* -> **Nisam novinar.**
9. *Se zovem Miloš.* -> **Zovem se Miloš.** (Sentence cannot start with 'se' or enclitic).
10. *Ne ja govorim srpski dobro.* -> **Ja ne govorim srpski dobro.**

## Verb Types Reference (Prezent Glagola -I/-E)

(Source: Image 4, Chart)

- **-I verbs** (raditi, učiti, govoriti, živeti, voleti) typically have stems ending in -i in the present (radi-, uči-, govori-, živi-, voli-).
- **Endings**:
  - Ja, Ti, On... -> -im, -iš, -i
  - Mi, Vi, Oni... -> -imo, -ite, -e (Wait, Image 4 chart shows endings `-e` for Oni?)
  - *Correction from chart:*
    - Oni rade (raditi -> rad-e)
    - Oni uče (učiti -> uč-e)
    - Oni govore (govoriti -> govor-e)
    - Oni žive (živeti -> živ-e)
    - Oni vole (voleti -> vol-e)

*Important:* The 3rd Person Plural (Oni) often ends in **-e** for these "i-stem" verbs in the present tense, unlike the infinitive vowel suggests.
